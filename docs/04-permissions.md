# Permissions

Permissions are role-based and tenant-scoped.

Initial permission groups:

- Residents.View
- Residents.Manage
- Medications.View
- Medications.Administer
- Nutrition.View
- Nutrition.Manage
- Rooms.View
- Rooms.Manage
- Finance.View
- Finance.Manage
- Reports.View
- Staff.View
- Staff.Manage
- Settings.Manage

Every important operation must create an audit log record.
