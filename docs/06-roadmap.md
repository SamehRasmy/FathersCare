# Roadmap

## Phase 0 - Project Foundation

Solution, projects, Clean Architecture, authentication, roles, permissions, tenant, audit, layout, and base design system.

## Phase 1 - Residents Core

Resident creation, basic profile, contacts, documents, medical profile, measurements, timeline, and rooms.

## Phase 2 - Medication Core

Resident medicine registration, shelf location, dose schedules, meal timing, administration log, missed doses, stock, and alerts.

## Phase 3 - Nutrition and Kitchen Core

Diet types, meal plans, special meals, meal distribution, kitchen stock, shortage tracking, waste, and medicine meal conflicts.

## Phase 4 - Dashboard

Priority now, room map, live alerts, today's schedule, daily summary, and latest activity.

## Phase 5 - Reports

Operational reports, financial reports, KPIs, PDF, and Excel.

## Phase 6 - Pilot

Real facility testing, fixes, UX improvements, permissions tuning, and report refinement.
